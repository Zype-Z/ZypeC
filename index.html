What is ZypeC

ZypeC (Zype Compilers) is a Cross-Platform Cross-Compiler Collection by Zype, Inc. We believe that great things can be done with OpenSources. So we made ZypeC OpenSource. You can get project details in GitHub. It is based on Python.

About the Project

The Project ZypeC (formerly TGCC) is the Cross-Platform Cross-Compiler Collection by Zype, Inc. (A part of TechGeeks). Currently it only supports Markdown to HTML compilation.

Installation

In this section you will know how to install ZypeC in you PC

Using PIP

The easiest way to install ZypeC is to run pip install ZypeC.

Using Wheel File

The second easiest and Offline way to install ZypeC is to download ZypeC from here and run pip install <Downloaded-File>.

Using legacy setup.py install

Download the ZIP from here, UnZIP it and run python setup.py install.

Getting Started

In this section you will know how to start your ZypeC journey.

Start with zype.config.json file

To get started you need to create a new file named zype.config.json with the content:

{
    "compiler": "compiler.json"
}

Create compiler

Create a file name based on the file you had written zype.config.json. And add the following data to it:


{
    "file": "File to be Compiled.md",
    "Content-Type": "text/markdown",
    "to": "html"
}

Start Compilation!

Now start Compilation with the command:


zype start
